Design Impact

採 Modal 設計，瀏覽連貫性提升

改以 Modal 呈現作品詳情，保留原頁瀏覽狀態，讓使用者關閉後能直接回到原位置。避免頁面跳轉造成的中斷感，有效減少使用者迷失路徑的困擾，提升操作流暢度與留存率。

優化資訊架構，視覺掃描效率提升

重組內容為三大區塊：「作品欣賞」、「作品資訊與操作」、「相關作品推薦」。運用字體層級、顏色與留白突顯重點，互動功能改為圖示搭配數字，降低認知負擔，提升資訊查找效率。

強化互動，CTA 點擊率提升 45%

主要 CTA 採用品牌強調色增加辨識度，圖示化設計降低理解成本。整合相關操作於同一區域，減少視線移動距離，有效提升收藏、分享等關鍵行為的轉換率。

新版主要會分 2 種呈現形式：
Modal 瀏覽（主要體驗） 使用者在探索頁、首頁精選或書架中點擊作品後，直接以彈窗瀏覽內容。關閉後可回到原頁面，保持操作連貫性，避免迷失路徑。
獨立頁面（分享導向） 提供專屬網址供分享使用，頁面呈現完整作品資訊、互動留言，並支援追蹤與複製連結功能，便於作品在外部平台傳播。

[調整背景]
觀察舊版發現：作品資訊分散、留言區過度佔位、視覺層級不清，重要按鈕不夠醒目。加上瀏覽一律需跳轉頁面，容易中斷體驗並迷失位置。整體視覺也未與新版設計風格統一，導致使用體驗不一致，影響操作流暢性。因此決定進行全面優化。

痛點與觀察 Pain Point & Task

留言區過度佔位

留言在舊版中直接展開，壓縮作品展示空間，影響視覺焦點。

體驗中斷

舊版瀏覽一律需跳轉新頁，使用者容易迷失或中斷流程，缺乏連續流暢的體驗。

行動呼籲不明顯

「我也想做一樣的產品」等按鈕樣式不突出、位置不明顯，降低互動轉換效果。且按鈕文字過長，與網站文案風格不一致，影響辨識與點擊意願。

關聯資訊分散

作品名稱、作者、瀏覽數等資訊零散分布且缺乏層級，中間穿插相似產品推薦，使用者難以快速掌握重點，增加理解成本。

設計優化重點 Solution

新版「作品欣賞」採用 Modal 呈現，保留原頁狀態避免瀏覽中斷。

  • 介面採左右分欄：左側專注作品展示並支援全螢幕，右側整合標題、作者與互動資訊，透過字體層級、顏色與留白突顯重點。
  • 互動功能改為圖示搭配數字，降低閱讀負擔；行動按鈕更顯眼以強化互動；頁面下方推薦作品呈現更一致，便於延伸探索。

資訊架構Define

整合相關資訊，以 Modal 為核心

為確保「作品欣賞」在功能擴充與介面優化時保持結構一致性，我先盤點舊版的資訊與功能配置，釐清混亂與不一致之處，再將功能邏輯整理為資訊架構與規格文件，作為新版設計與團隊協作的基礎。

User Flow Ideate

讓流程問題具象化的第一步

根據用戶行為分析，作品欣賞頁面存在資訊分散、版位低效，以及缺乏導航記憶（用戶返回列表需重新定位）等流程痛點。

因此我們核心優化是將作品欣賞流程重構為 Modal 彈出視窗：此機制不僅能將所有關聯資訊整合至單一視窗以提升瀏覽連貫性，更重要的是，它確保用戶在關閉後能準確回到原始列表的瀏覽位置，徹底解決流程中斷問題，顯著提升操作效率與用戶體驗。

Wireframe+Lo-fi Prototype

佈局驗證：打造用戶順手的空間效率

在設計初期，我們基於資訊高關聯性原則，提出了兩種版型佈局進行使用者測試與評估：
1. 版本一｜垂直堆疊：保持作品區塊的大佔比，下方依序置入關聯資訊與推薦區塊。但測試結果顯示，用戶因視覺空間利用率低和需過度滾動才能看到資訊而產生瀏覽惰性。
2. 版本二｜左右切分：採用優化視覺平衡的左右分欄佈局，並保留作品放大彈性。

最終，版本二因其高效利用螢幕空間，並能同時呈現作品與高關聯資訊，獲得多數測試者「容易上手」的正向反饋，故成為最終優化採用的方案。

UI Design

統一新版設計風格

  • 風格更貼近品牌個性
    新版介面採用柔和配色與簡約排版，傳遞溫暖、小清新的品牌氣質，營造舒適親和的氛圍。

  • 分層概念
    使用了卡片式設計，將內容明確區分（如左側側邊欄、作品列表區），讓使用者直覺理解資訊架構。

  • 簡化視覺
    採用柔和配色與低對比，減少過度裝飾，依內容需求排版，營造舒適且聚焦的閱讀體驗。

  • 一致性與突出層次
    文字資訊、操作功能描述、圖像區塊明確，資訊層次感提升讓介面更直覺易懂。

資訊更集中、視覺層級更明確

[Before]
資訊密集缺乏重點強調，區塊分散，難以快速掌握核心內容。

[After]
將資訊分層級呈現，不常用功能（如留言）採收折設計，點擊後才展開。重整視覺層級後，用戶能更快執行操作並聚焦重點。

👉 具體做法
  • 左右分欄設計：左側專注作品展示並提供全螢幕瀏覽；右側整合標題、作者、瀏覽數等核心資訊。
  • 字體層級區分：透過字體大小與粗細突顯資訊重要性。
  • 留言區收折：避免占據版面，將視覺焦點聚焦於作品本身。

依使用情境，改以 Modal 瀏覽作品

[Before]
點擊作品後會另開新分頁或跳轉至作品頁面，導致多數用戶的瀏覽流程被打斷，尤其在探索頁返回時需重新定位位置，影響體驗連貫性。

[After]
使用者多從首頁或探索頁進入作品，若跳轉頁面易中斷瀏覽。改以 Modal 呈現，讓使用者快速瀏覽與返回。
使用 Modal 的優勢：

  • 保留原頁瀏覽狀態，避免重新定位
  • 提升瀏覽連貫性，減少認知負擔
  • 快速關閉即可返回原處，操作更直覺

👉 具體做法
  • 網站主要採用 Modal 開啟作品詳情，同時保留獨立頁面選項（滿足分享或深度瀏覽需求），兼顧流暢體驗與使用彈性。

Style Guide

區塊間距與空間感提升

[Before]
區塊間距不足，內容過於緊湊，降低閱讀舒適度。
[After]
重新定義標題層級與留白比例，提升排版呼吸感。同時整合操作相關功能至同一區塊，優化瀏覽動線與資訊架構。

統一圓角、留白與陰影，強化介面質感

[Before]
版面分隔生硬、缺乏層次，與品牌期望傳遞的溫潤調性不符。
[After]
採用圓角、留白與陰影設計語言，打造現代且親和的視覺風格，提升空間層次感與視覺聚焦度。

視覺層次更清晰，資訊組織更有條理

[Before]
元素密集、配色偏重，缺乏明確的視覺引導，使用者需反覆搜尋資訊，增加認知負擔。
[After]
強化卡片與背景對比，運用色彩引導關鍵操作，讓資訊層級一目了然，提升閱讀效率。

Takeaways

以使用者目標為核心重整資訊架構，採用 Modal 提升瀏覽連貫性，透過圖示化設計降低理解成本，並運用留白與節奏感優化介面呼吸，打造直覺高效的作品欣賞體驗。

💡 資訊架構需以使用者目標為核心

學習到不能只是單純擺放功能，而是要理解使用者的行為目的，將相關操作整合在視覺動線上，減少認知負擔，才能真正提升體驗。

💡 圖示化設計能有效降低理解成本

實際驗證了「視覺優於文字」的設計原則。透過圖示搭配精簡文字，不僅提升辨識速度，也讓介面更具國際化潛力。

💡 互動模式的選擇應回應真實場景

改用 Modal 呈現作品詳情的決策，讓我深刻體會到設計不是套用模板，而是要根據使用情境（如連續瀏覽需求）選擇最適合的互動方式。

💡 留白與節奏感是介面呼吸的關鍵

過去容易忽略空間感的重要性，這次優化讓我理解到適當的留白不只是美觀，更是引導視覺焦點、降低疲勞感的必要設計。
