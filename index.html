UI DESIGN PROJECT

Editor2｜作品線上編輯器

整體優化：打造更直覺流暢的編輯體驗

Editor2 作為點點印全產品的線上編輯器，支援用戶在瀏覽器中輕鬆製作相片書、邀請卡、婚禮書約等個人化商品。透過拖拉照片、編輯文字與自訂版型，使用者能快速完成設計並直接下單印製。 我負責此產品的整體 UI UX Redesign，從編輯介面到操作流程，著重提升使用的直覺性與彈性，並優化跨裝置體驗。

Single｜輕量作品編輯器

聚焦效率與易用性，重新定義快速製作流程

為因應不同產品特性，Single 編輯器聚焦「簡單快速、直覺上手」，主打極簡流程與自動排版，與 Editor2 的高度自訂形成互補。我主導其 UI/UX Redesign，優化操作動線與頁面引導，並強化跨裝置一致性，讓更多用戶能快速完成設計，提升作品完成率與下單轉換率。

TinTint Projects｜UI Enhancement

點點印品牌官網 多頁面的 UI 優化專案

我參與官網多個核心頁面的 UI 優化設計（如「我的作品、作品列表、帳號設定、月曆系列產品、周邊配件、全產品一覽...」等），這些頁面與會員操作及購買決策密切相關。設計上著重於資訊清晰、視覺引導與轉換效率。我負責頁面改版與元件規劃，整合使用者觀察與內部回饋，優化內容結構與操作流程，並同步進行視覺統一與行動裝置優化，提升品牌一致性與跨裝置體驗。

LUCY｜虛擬面試平台

Simple Video Interview Software

LUCY 是一款整合徵才流程的 SaaS 產品，透過虛擬面試功能，協助面試官與應徵者在不同時間與地點完成面試，有效降低人力與時間成本。我負責官網與軟體介面的設計，並主導品牌識別與行銷素材製作，涵蓋 Logo、視覺風格與平面設計。介面風格以現代俐落與易用為核心，優化操作流程並提升用戶的面試體驗。
