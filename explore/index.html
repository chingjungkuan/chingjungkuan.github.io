Design Impact

清晰的資訊架構

透過重新分層，將頁面劃分為「精選作品」與「主題探索」兩大區塊，並搭配卡片式設計，讓使用者更直覺理解內容結構。

不中斷的探索體驗

改用 Modal 瀏覽，頁面返回時的重複操作減少，觀察 GA 後台發現使用者平均停留時間增加 30%。

更具啟發性的內容呈現

卡片新增瀏覽數據，並針對不同主題預設合適的產品（如婚禮 → 映畫本），提升探索動機，為缺乏靈感的使用者提供明確方向。

概述 Overview

點點印｜探索好作品

#B2C #RWD #Redesign

探索好作品 頁面是展示點點印用戶透過線上編輯器創作的相片書與設計作品的展示空間。不僅精選了多樣化的優質範例，幫助沒有靈感的使用者獲得啟發，也提供了主題與產品類型的篩選功能，讓使用者能依照需求快速找到對應的內容，進一步激發創作靈感。

[調整背景]
舊版頁面存在分類重疊、資訊不足與瀏覽中斷等問題，導致使用者難以快速找到想看的內容，也降低了探索與獲得靈感的效率。因此需要透過資訊架構與互動設計的調整，改善整體體驗。

痛點與觀察 Pain Point & Task

🗯️ 分類與篩選結構易混淆

  • 左側篩選清單項目過多且層級不清，使用者容易忽略或不知如何開始；篩選後的狀態也不明顯，缺少高亮或快速清除等即時回饋。
  • 主題 Icon 按鈕偏灰，缺乏吸引力與選中狀態，降低互動與辨識度。

🗯️ 導覽語意不明確與資訊不足

  • 每張卡片只看到縮圖和作品標題，沒有額外的敘述引導或互動，導致使用者難以判斷是否想點進去。
  • 頁面標題「大家的點點印作品」過於籠統，對第一次進來的使用者沒有強烈指引。

🗯️ 流程與版面影響探索

  • 作點擊作品卡片後會跳轉至作品欣賞頁，返回列表時需自行尋找原本位置，導致瀏覽流程被中斷。
  • 卡片間距不一致，作品圖片偏小，版面顯得擁擠，降低瀏覽舒適度。

🗯️ 排序與分頁的探索效率低

  • 「熱門 / 最新」混在在側邊導覽列，且看後台實際使用此功能佔少數。
  • 分頁需要手動點擊跳轉下一個分頁，探索的流暢度不如同頁面載入。

設計優化重點 Solution

  • 瀏覽作品從「另開新分頁」改為「 Modal 顯示」，減少中斷，讓體驗更連續。
  • 重新設計篩選與主題分類的層級結構，幫助有具體需求的使用者更快找到目標作品。
  • 作品卡片新增瀏覽數這類的互動數據，提升探索動機與參與感。
  • 改用分批載入作品列表，減少初始讀取量，提升效能並讓瀏覽更流暢。
  • 加入官方精選作品，為缺乏靈感的使用者提供創作方向與啟發。

延伸閱讀｜作品欣賞 Modal 專案

資訊架構Define

重構資訊架構，建立清晰的探索層級

  • 頂部以「探索好作品」+ 副標「點點印 精選」明確告訴使用者這裡是「推薦作品」的區塊，比原本模糊的「大家的作品」更有指引性。
  • 先呈現「精選作品」輪播，再往下才是主題分類探索，層級清楚。

User Flow Ideate

視覺化流程，釐清痛點並重構架構

我先繪製了 User Flow，將頁面路徑與層級視覺化，協助團隊快速辨識舊版痛點。例如：篩選功能在全展開時容易造成混淆；主題篩選區塊缺少「全部」標示，讓使用者不清楚當前瀏覽的是「全部主題」；產品篩選則因預設收合與樣式不明確，造成使用上的困擾。 根據這些洞察，先行優化介面結構與資訊架構，使層級更清晰、流程更直覺，再進一步進入設計與開發階段。

Wireframe+Lo-fi Prototype

以低擬真原型快速驗證，化繁為簡，確保探索流程更直覺

我們觀察到使用者在「篩選時常出現遲疑」，顯示操作流程不夠直觀。因此以低擬真原型快速建立 Wireframe，協助團隊對齊元件配置並驗證能否降低思考負擔。 優化後將功能依層級清楚區分，並僅在需要時透過 Modal 呈現細節，使介面更簡潔且瀏覽不中斷。

UI Design

統一新版設計風格

  • 符合新版品牌風格與氛圍
    新版介面採用柔和配色與簡約排版，傳遞溫暖、小清新的品牌氣質，營造舒適親和的氛圍。

  • 清晰層級設計
    從資訊分層到視覺配置，將內容劃分為「推薦」與「主題探索」兩大區塊，並以卡片式設計與明確標題，讓結構直觀且易於理解。

  • 細節亮點設計
    在精選標題加入 ✨ 圖示，強化「精選」的特別感，並透過細節提升瀏覽體驗。

精選定位更清晰

[Before]
資訊密集，缺乏重點強調，區塊層級不易辨識。

[After]
以「探索好作品」+「點點印 精選」明確定位，先透過精選輪播吸引視線，再引導至主題分類，層級清晰、探索更直覺。

👉 具體做法
  • 強化導向感：標題與副標凸顯頁面定位。
  • 集中注意力：精選輪播吸引視線，帶動瀏覽節奏。
  • 提升結構感：形成由上而下的清楚層級。

主題篩選更直覺

[Before]
左側長清單繁雜，上方主題按鈕狀態不明確，分類層級混亂，使用者難以上手。

[After]
改為「今天想看什麼主題」的口語化呈現，並以水平 icon 按鈕設計；同時結合「篩選」按鈕，點擊後展開 Drawer，提供主題＋產品的多條件篩選，讓操作更輕量直觀。

👉 具體做法
  • 提升辨識度：水平排列的 icon 按鈕更直觀。
  • 降低操作成本：將「全部」與各分類並列，減少學習負擔。
  • 增強探索效率：簡化層級，快速切換與瀏覽。

作品卡片資訊更豐富

[Before]
卡片僅有縮圖與標題，缺乏作者與互動數據，使用者判斷依據不足。

[After]
新增「觀看數」icon 與數字，補充更多資訊，讓卡片結構更完整，瀏覽判斷更有依據。

👉 具體做法
  • 提升探索動機：觀看數作為熱門指標，增加使用者點擊意願。
  • 增強資訊完整性：卡片包含標題、作者與互動數據。
  • 改善一致性：各卡片資訊呈現統一，降低理解成本。

互動與操作體驗提升

[Before]
分頁設計需透過 Page Control 逐頁切換，導致探索中斷；一次載入過多資料也造成初始載入緩慢，瀏覽卡頓。

[After]
內容達到區塊上限時改為顯示「載入更多」的按鈕設計，讓探索流暢不中斷，並透過分批載入縮短初始時間、降低伺服器與前端負擔。
精選作品以輪播方式呈現，減少佔用空間，卻能展示更多內容。

👉 具體做法
  • 保持流暢性：使用者連續瀏覽不中斷。
  • 優化效能：分批載入避免前端卡頓，縮短初始時間。
  • 提升展示效率：精選作品輪播呈現更多內容，節省版面。

情境導向的多樣化預設

[Before]
舊版主題與產品的篩選較為分散，使用者需自行組合條件，容易增加思考成本。

[After]
依主題自動帶入最相關的產品（如旅遊→寫真本、婚禮→映畫本），讓使用者更快看到符合情境的作品範例。

👉 具體做法
  • 降低思考成本：自動帶出最適合的產品類型。
  • 增強探索啟發：主題對應產品，更容易獲得靈感。
  • 提升導向性：提供個人化、情境化的入口。

視覺層級與排版優化

[Before]
分區與層級不明顯，icon 樣式偏灰，互動感不足；卡片排列間距不一致，整體視覺顯得凌亂。

[After]
分區標題清楚，留白適切；icon 與文字狀態一致，互動感明確；卡片間距整齊，排版更乾淨。

👉 具體做法
  • 提升辨識度：區塊標題與留白讓層級更清楚。
  • 強化互動性：icon 狀態清晰，點擊更直觀。
  • 增進整體觀感：統一的排列與間距帶來乾淨專業的版面。
Style Guide

區塊間距與空間感提升

[Before]
區塊間距過小，內容顯得緊湊，降低閱讀的舒適度與辨識度。
[After]
透過重新定義標題層級與留白，讓排版更清晰，同時整合篩選內容於同一區塊，強化資訊結構與閱讀體驗。

統一圓角、留白與陰影，強化介面質感

[Before]
卡片與版面分隔生硬，缺乏柔和層次；icon 按鈕缺乏一致規範，狀態不明顯，使用者不易辨識可點擊性。
[After]
統一按鈕與 icon 規範，並導入圓角、留白與陰影設計，營造現代化且友善的風格，提升空間感與聚焦度。

視覺層次更清晰，資訊組織條理化

[Before]
作品卡片比例偏小，排版顯得擁擠，重點難以凸顯。
[After]
調整卡片比例與排版規則，使區塊更一致，資訊層級更清晰易讀。

UI Kit

Takeaways
整體以使用者瀏覽行為為核心，透過資訊分組、視覺層級與細節優化，打造直覺、流暢且易理解的作品欣賞體驗，同時確保設計可落地於團隊實作。

💡 以使用情境為核心思考

透過觀察使用者在「探索作品」的瀏覽行為，我理解到設計不只是呈現資訊，而是要維持流暢、不中斷的體驗，Modal 的引入就是基於這個思考。

💡 資訊層級與分組的重要性

整理作品資訊的歸納分組，讓我更清楚視覺層級如何影響使用者理解，也培養我用簡單設計語言引導注意力的能力。

💡 細節決定使用感受

從字體大小、顏色到圖示呈現，每個小調整都能降低使用者認知負擔，讓我體會到「細節優化」對整體體驗的重要性。

💡 設計需兼顧團隊與可實作性

在規劃資訊架構與 Modal 流程時，我學會如何將抽象想法整理成可落地的規格文件，讓團隊能快速理解與實作，強化了自身跨角色協作的能力。
