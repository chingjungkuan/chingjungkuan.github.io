FAQ

Q：可以只做 UI，不做 UX 研究嗎？

可以，範圍可以切。有現成的 UX 規格或 wireframe 給我，我接著做視覺也沒問題。

Q：修改幾次？

專案制通常包含 2 輪大改。超過的話以時薪計算，事先會說清楚。

Q：交付什麼格式？

Figma 設計稿，含 Component 標注與切圖規格。有需要可以附 HTML / CSS 實作。

Q：可以簽 NDA 嗎？

可以，提供合約範本或你方合約皆可。
